ROBOTERRY

To design an autonomous manual robot controlled by mobile (only using Bluetooth ) which will be capable of running away from and chasing the other robot in an arena having various obstacles AND also capable of getting the various point by crossing some area.

Details

EVENT DESCRIPTION


Get ready to experience the thrill and joy of the astounding game of run and chase. Where robots will obey your orders religiously.
▪It is a one-on-one versus mode game where one robot will run away & collects points and other has to chase & catch it before it reaches the end-point and also has to stop the runner to getting the points means runner can grab or go through some extreme checkpoints where the bot will get some extra point
▪There will be two sub-rounds for each match between bots namely ‘A’ and ‘B’ Each Bot will have to run and chase alternatively
▪At first round ; If bot ‘A’ will RUN and ‘B’ will CHASE.
▪At second round; Bot ‘A’ and Bot ‘B’ will be change it runner to chaser
▪Motive of the chasing bot will be to touch the bot running away before it reaches the end-point and to stop the bot to getting points.
▪During each round if the RUNNING bot touches the equilibrio symbol ; then chasing bot will have to touch the symbol before it continues its chase again after ‘A’

ARENA DESCRIPTION :

The arena consist of a plywood base having an area 8*8 feet.There will be two openings at the two diagonally opposite ends of the arena for the entry of the bots.At centre is equilibrio symbol located.Many hurdels are inside the Arena.Arena also have some checkpoints where runner bot can points some score. Width of the path is 1ft .Arena will be fenced from all four sides except the opening provided for the entry.

Rules

Event of Cultural

    Round 1 :


    ▪The team has to be present at the event place within 5 minutes of final call otherwise they will be disqualified.
    ▪Team also have to test her bot in testing arena only then they will allow to start the game
    ▪One semi-autonomous bot per team is allowed.
    ▪The gameplay will start only after the signals of organisers.
    ▪After the bot starts, no team member is allowed to touch it.
    ▪Each team will have to run and chase alternatively.
    ▪Maximum 5 minutes will be given for technical time out. Afterthat if bot takes above 5 minutes then team will be disqualified.
    ▪Disqualified team will maybe get a chance of wild card entry only if there bot work properly and organisers have time.
    ▪The organisers reserve the rights to change any or all of the above rules as they deem fit. Change in rules, if any will be highlighted on the website and notified to the registered teams.

    JUDGING CRITERIA :


    ▪The runner team will be awarded 100 points for successfully completing the Arena.
    ▪The runner team will be awarded 5/10/20/50 points for going through some extreme point arena.
    ▪The runner team losses 5 OR 10 points for each time it touches the arena’s wall Or Hurdles.
    ▪The runner team losses 20 points for each time it crosses the equilibrio symbol. (However it is not mandatory to cross the equilibrio).
    ▪At the end the team securing the highest points would be the winner.
